# LangTutor Admin Console

A React + TypeScript + Tailwind admin dashboard for managing the LangTutor platform. Handles end users, admin users, roles, languages, personas, subscription plans, reports and audit logs. Talks to the FastAPI backend running on `http://localhost:8001`.

## Getting started

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` and proxies `/api` requests to the backend on port 8001.
